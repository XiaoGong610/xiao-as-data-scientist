# xiao-data-scientist

Claude Code setup for data science workflows.

## Setup

1. Clone this repo
2. Open it in Claude Code: `cd xiao-data-scientist && claude`

## Custom Commands

| Command | Description |
|---------|-------------|
| `/eda` | Exploratory data analysis on a dataset |
| `/clean` | Clean and preprocess a dataset |
| `/profile` | Generate a detailed data profile report |

## Structure

```
CLAUDE.md               # Instructions for Claude Code
.claude/commands/       # Custom slash commands
templates/              # Reusable templates
data/                   # Local data files (gitignored)
output/                 # Analysis outputs (gitignored)
```
